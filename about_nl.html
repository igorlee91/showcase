Over mij

WIE BEN IK?

Na lange tijd gewerkt te hebben in diverse sectoren, heb ik mijzelf eindelijk teruggevonden in web ontwikkeling.

In november 2019 ging mijn opleiding voor PHP ontwikkelaar van start bij de VDAB (campus gelegen in Wevelgem). Ik heb op dit moment een goed basiskennis van HTML/CSS, alsook Javascript en ik blijf constant doorgroeien.

In de loop van mijn studietraject heb ik een passie ontwikkeld voor het UX/UI design. Werken aan de user experience boeit mij enorm en doet mij willen bijdragen.

WAT KUNT U VERWACHTEN?

Naarmate mijn kunnen doe ik er alles aan om uw project binnen de afgesproken deadline te realiseren.
